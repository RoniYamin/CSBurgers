# CS Burger

## Overview
This project is a full-stack web application, following the **MVC** (Model-View-Controller) design pattern, designed for managing hamburger orders and deliveries.

## Technologies Used  
🍔 **Node.js** / **Express**  
🍔 **MongoDB**  
🍔 **Socket.io**  
🍔 **jQuery** / **AJAX**  
🍔 **HTML5** / **CSS3**
